# 🎨 Random Color Generator (React)

A simple and interactive React application that generates random HEX and RGB colors. Users can switch between color formats and instantly apply them as the background.

---

## 🚀 Features

- 🎲 Generate Random HEX Colors
- 🌈 Generate Random RGB Colors
- 🔄 Switch between HEX & RGB modes
- 🎨 Dynamic background color update
- ⚛️ Built using React Hooks (useState)

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- CSS

---

## 📸 Preview

The background color changes dynamically with each click, displaying the generated color code on screen.

## ⚙️ Installation

1. Clone the repository:

```bash id="proj2"
git clone https://github.com/your-username/random-color-generator.git
```

2. Navigate to project folder:

```bash id="proj3"
cd random-color-generator
```

3. Install dependencies:

```bash id="proj4"
npm install
```

---

## ▶️ Run the App

```bash id="proj5"
npm start
```

App will run on:

```
http://localhost:3000
```

---

## 💡 How It Works

- HEX colors are generated using a predefined array of values (0–F)
- RGB colors are generated using random numbers (0–255)
- React state (`useState`) updates the background dynamically
